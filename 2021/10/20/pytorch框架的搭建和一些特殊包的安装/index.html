软件

pycharm anaconda cudnn cuda

步骤

1.Anaconda创建虚拟环境



此时在Environments选项卡中，已经出现刚刚创建的虚拟环境 【注】才创建的虚拟环境只包含最基本的库

2.pycharm创建conda工程


在Anaconda安装文件夹下找到envs文件夹，里面存储的都是你创建的虚拟环境，然后选择刚刚创建的虚拟环境文件下的python.exe，点击OK


创建完成后自动进入如下界面：

3.cudnn&cuda

Cudnn和cuda对版本要求比较高，必须根据自己电脑对应的驱动版本进行下载（还没找到展示比较全面准确的cuda和cudnn版本对应的表，对着官网的提示下载即可）

Cuda和cudnn注册登录很耗时间，网盘自取
cuda
链接：https://pan.baidu.com/s/1WjfSWtSBglCF6C-mO8dz9Q
提取码：73xw
Cudnn:
链接：https://pan.baidu.com/s/1PUs_jxuVcJ4Txh7o-blsyg
提取码：topp
有一次cuda装了一般才发现没装驱动，所以我把驱动的网盘也放下面了，
链接：https://pan.baidu.com/s/1SJXh9ApG8VWwdVHcHjTuhw
提取码：h4fv

actiavate 文件名：激活虚拟环境，进入虚拟环境中
conda config –add channels:这个命令可以换源到镜像网站下载，缩短下载的时间
conda install pytorch==1.2.0 torchvision==0.4.0 cudatoolkit=10.0：安装对应的包
最后下载对应的包

Cuda安装成功验证

Nvcc -V (V大写)

Cudnn安装成功验证

依次执行：
cd C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v10.0\extras\demo_suite
bandwidthTest.exe

Result=PASS即可

运行：deviceQuery.exe

Result=PASS即可

4.安装pytorch，在虚拟环境中安装对应的包

结束

下载torch和torchversion

pytorch导入后在python中导入时显示为torch
pytorch和torchversion下载地址.

pip install 对应文件的全名（注意要把后缀名也复制进去）

上述方法不成功的话可在命令行下尝试运行下面命令

#安装pytorch命令
conda install pytorch

#安装torch命令
conda install torchvision -c pytorch #安装torch命令

最后验证安装的pytorch，torchversion
